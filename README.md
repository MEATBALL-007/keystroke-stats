# keystroke-stats

หน้าสรุปงานสถิติ Keystroke — การหาค่ากลาง 5 แบบ (Mean, Weighted Mean, Combined Mean, Median, Mode)
จากข้อมูลการกดปุ่ม 10 วัน (2026-06-21 ถึง 2026-06-30) พร้อมวิธีทำครบทุกขั้นตอน

ไฟล์เดียว self-contained: HTML/CSS/JS + MathJax (CDN) ไม่มี backend
JS คำนวณค่าเองจากข้อมูลดิบและ verify ตรงกับคำตอบที่กำหนด

👉 เปิดดู: https://meatball-007.github.io/keystroke-stats/
